# نظام إدارة المستخدمين للمدير

## الميزات الجديدة المضافة

### 1. تبويب إدارة المستخدمين (المدير فقط)
- تبويب جديد "إدارة المستخدمين (المدير)" متاح للمديرين فقط
- واجهة شاملة لإدارة جميع المستخدمين في النظام

### 2. إحصائيات المستخدمين
- إجمالي عدد المستخدمين
- عدد المستخدمين النشطين
- عدد المديرين
- عدد المشرفين

### 3. وظائف إدارة المستخدمين

#### إضافة مستخدم جديد
- الاسم الكامل
- اسم المستخدم (فريد)
- البريد الإلكتروني
- رقم الهاتف (اختياري)
- كلمة المرور مع تأكيد
- نوع الصلاحية (مدير/مشرف/مستخدم)
- الصلاحيات التفصيلية:
  - إضافة مراكز
  - تعديل مراكز
  - حذف مراكز
  - إضافة أفراد
  - تعديل أفراد
  - حذف أفراد
  - عرض جميع المراكز
  - عرض جميع الأفراد

#### تعديل المستخدمين
- تعديل جميع بيانات المستخدم
- تغيير كلمة المرور (اختياري)
- تعديل الصلاحيات التفصيلية
- تحديث نوع الصلاحية

#### حذف المستخدمين
- حذف المستخدم نهائياً من النظام
- تأكيد قبل الحذف

#### تفعيل/إيقاف المستخدمين
- تبديل حالة المستخدم (نشط/غير نشط)
- منع تسجيل الدخول للمستخدمين المتوقفين

### 4. قائمة المستخدمين
عرض تفصيلي لجميع المستخدمين يتضمن:
- الاسم الكامل
- اسم المستخدم
- البريد الإلكتروني
- رقم الهاتف
- نوع الصلاحية
- حالة الحساب (نشط/غير نشط)
- آخر تسجيل دخول
- تاريخ إنشاء الحساب
- أزرار الإجراءات (تعديل/حذف/تفعيل-إيقاف)

### 5. الأمان والصلاحيات
- الوصول مقصور على المديرين فقط
- فحص الصلاحيات قبل عرض التبويب
- تشفير كلمات المرور
- التحقق من صحة البيانات المدخلة

### 6. واجهة المستخدم
- تصميم متجاوب
- ألوان مميزة للشارات (نشط/غير نشط)
- أزرار صغيرة للإجراءات
- شبكة منظمة للصلاحيات التفصيلية
- تنبيهات واضحة للعمليات

## كيفية الاستخدام

1. **الوصول إلى إدارة المستخدمين:**
   - تسجيل الدخول كمدير
   - الانتقال إلى تبويب "إدارة المستخدمين (المدير)"

2. **إضافة مستخدم جديد:**
   - النقر على "إضافة مستخدم جديد"
   - ملء جميع البيانات المطلوبة
   - تحديد الصلاحيات المطلوبة
   - النقر على "إضافة المستخدم"

3. **تعديل مستخدم:**
   - النقر على "تعديل" بجانب المستخدم
   - تعديل البيانات المطلوبة
   - النقر على "تحديث المستخدم"

4. **حذف مستخدم:**
   - النقر على "حذف" بجانب المستخدم
   - تأكيد الحذف

5. **تفعيل/إيقاف مستخدم:**
   - النقر على "إيقاف" أو "تفعيل" بجانب المستخدم
   - تأكيد العملية

## ملاحظات مهمة

- جميع العمليات تتطلب صلاحيات المدير
- كلمات المرور يجب أن تكون 6 أحرف على الأقل
- اسم المستخدم يجب أن يكون فريداً
- يمكن إيقاف المستخدمين بدلاً من حذفهم للحفاظ على البيانات
- جميع العمليات مسجلة في النظام
